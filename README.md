# Smart Dress Tracking & Outfit Recommendation System

This project is a mobile application designed to help users organize their wardrobe, plan outfits, and build smarter personal dressing habits.

## Project Structure

- `src/components`: Reusable UI components.
- `src/screens`: Application screens.
- `src/navigation`: Navigation configuration.
- `src/services`: API and business logic services.
- `src/hooks`: Custom React hooks.
- `src/constants`: App constants (colors, strings, etc.).

## Getting Started

1. Install dependencies: `npm install`
2. Start the development server: `npm start`
